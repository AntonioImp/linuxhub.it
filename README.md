## Installation
### Dependencies
- ruby
- ruby-devel
- github-pages
- jekyll-paginate

```bash
gem install github-pages
gem install jekyll-paginate
```

### Build
```bash
jekyll build
```

### Server
```bash
jekyll serve
```

This also automatically builds the project.
