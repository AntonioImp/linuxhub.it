## Installation
### Dependencies
- ruby
- ruby-devel
- github-pages
- jekyll-paginate-v2

```bash
gem install github-pages
gem install jekyll-paginate-v2
```

### Build
```bash
jekyll build
```

### Server
```bash
jekyll serve
```

This also automatically builds the project.
